# v1.0 — Slow mechanic for shells

## Context

Shells currently cause the hit player to discard a card. This is being replaced with a new **slow counter** mechanic: shell hits now add a slow counter instead, and slow counters reduce movement by 1 per counter (minimum 1 move per turn). This makes shells a strategic debuff rather than card loss.

### Slice 7 — Player UI: dark mushroom icons on cards when slowed

**File:** `client/src/player_cards.js`

When the player has slow counters AND `hasMovedThisTurn` is true, ALL mushroom icons on cards in hand should display as dark mushrooms instead of regular mushrooms.

Add a function `updateCardMushroomIcons(useDark)` that iterates all cached card elements and swaps mushroom icon `src` between `/mushroom.svg` and `/dark_mushroom.svg`:

```javascript
export function updateCardMushroomIcons(useDark) {
  const src = useDark ? "/dark_mushroom.svg" : "/mushroom.svg";
  for (const [, el] of cardElements) {
    const items = JSON.parse(el.dataset.items);
    const icons = el.querySelectorAll(".card-item");
    items.forEach((item, i) => {
      if (item === "mushroom") icons[i].src = src;
    });
  }
}
```

**File:** `client/src/player.js`

In the `onStateChange` handler, when `me.hasMovedThisTurn` or `me.slowCounters` changes, call `updateCardMushroomIcons(me.hasMovedThisTurn && me.slowCounters > 0)`. Also reset on `cardsDrawn` (new turn).
