# v0.21 - River Rules

## Context

Currently any player can buy from any river regardless of their position in the race. This feature restricts river access based on rank: leading players can only buy from cheaper rivers, while trailing players have access to all rivers. This creates a catch-up mechanic. Designed to scale to a future 4th river.

## Rule

- Rank 1 → only river 0
- Rank 2 → rivers 0–1
- Rank 3 → rivers 0–2
- Rank 4+ → all rivers
- Rank 0 (unranked, all tied on start) → all rivers

Formula: `rank === 0 || rank >= riverCount || riverId < rank`
