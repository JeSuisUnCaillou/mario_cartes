# Plan: Display Players in Board

## Goal
Show all players in the board's top info bar, with their helmet, name, and card count.

## Slices

### Slice 1: Server sends hand count to board
- Add `handCount: p.hand.length` to `broadcastPlayers()` payload in `GameRoom.js`

### Slice 2: Split info bar into two zones and display players
- **CSS**: Restructure `.board-info-bar` into two rows:
  - Top row: title (+ QR on the left)
  - Bottom row: horizontal list of players
- **Board JS** (`createInfoBar`): Add a bottom row container for players
- **Board JS** (`updatePlayers`): Populate the players row with DOM elements:
  - Each player: name label on top, helmet image, row of card back images below
  - Disconnected players: `opacity: 0.5` (same rule as in-game helmets)
  - Card backs count matches `handCount` from server
