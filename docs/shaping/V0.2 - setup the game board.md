# V0.2 Setup the game board

I have made a svg of the racetrack and a helmet in /assets.

I want the game board to use [Phaser js](https://github.com/phaserjs/phaser) to create the game board and place the racetrack in the middle.

The racetrack data is represented in the game data by a chained list of 14 cells with ids 1 to 14. The first cell (the checkered one) has the property `finish_line: true`. The chained list loops : the 14th cell's next cell is the 1st cell.

Each new player is represented by a helmet placed on the checkered cell. 
There is a list of players in the game data. Each player have a "cell_id" representing their position on the racetrack.
They all start with a cell_id: 1.

When a player hits the ping button, he goes to his current cell's next cell. His position is updated in the game data and the board client should move him to the next cell.

We will replace the current board view in the client by a phaser scene.
