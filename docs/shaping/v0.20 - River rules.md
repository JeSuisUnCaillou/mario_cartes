# v0.20 - River rules

The first ranked player can only buy from the first river.
The second ranked player can only buy from the first and second river.
The third ranked player can only buy from the first, second and third river
The other players can buy from all rivers.

There will be a 4th river after, so implement it exactly like that.

In the UI, we will displayed under the price of each river :
 - river 1 : the 3 rank icons, normally
 - river 2 : the 1st rank icon in grey and barred in red, and the 2nd and 3rd icons normally
 - river 3 : the 1st and 2nd rank icons in grey and barred in red, and the 3rd icon normally
