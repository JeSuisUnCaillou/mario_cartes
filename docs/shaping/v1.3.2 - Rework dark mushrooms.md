# v1.3.2 - Rework dark mushrooms

## Rework of the mechanic

Let's rework the slow counters (also called "dark mushroom").

I want to make the dark mushroom simpler : remove the rule "except the first mushroom of the turn" in the rules, and also in the actual game mechanics. Now, each dark mushroom cancels exactly one mushroom, no exception.

The new rule is : Maximum 2 dark mushrooms per player.


## Correcly display dark mushrooms on cards with several mushrooms

In the player's hand, when we replace the mushrooms by dark mushrooms, if I have one dark mushroom and a card with 2 mushrooms, only the first mushroom must become dark on this card.

Other example: if I have a card with 3 mushrooms and I have 2 dark mushrooms, only the first 2 mushrooms of this card must become darK. And if I play another card with only one mushroom, then the card with 3 become 1 dark and 2 normal mushrooms.
