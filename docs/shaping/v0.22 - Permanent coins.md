# v0.22 - Permanent coins

I added on the racetrack json some permanent coins on cells. And a permanent coin asset.

When a cell has a permanent coin in the json, there is a permanent coin (blue coin) as the first element of the objects grid of the cell.

When a player reaches a cell with a permanent coin, he gains one permanent coin. This blue coin is added to the player's ui at the beginning of his coin list.

The permanent coin jumps and rotates twice when a player reaches his cell.

A player can spend a permanent coin like any other coin.

The spent permanent coins are restored when a player ends his turn.

Each player start with 0 permanent coins.
