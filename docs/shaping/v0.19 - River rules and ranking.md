# v0.19 - Live ranking

We are going to compute a live ranking, after every move.

All players start with an empty rank.
When the first player moves, his rank becomes 1, and all other player's rank becomes 2nd.
When any player moves, All the players ranks are computed based on their position on the board.
If two players are on the same cell, they have the same rank.

If we have this configuration on three cells :

1. player1
2. player2, player3
3. player4

then the rankings are : 

player1: 1
player2: 2
player3: 2
player4: 4


If we have this configuration on 3 cells :

1. player1
2. player2, player3, player4, player5
3. player6

then the rankings are : 

player1: 1
player2: 2
player3: 2
player4: 2
player5: 2
player6: 6

What I mean is that when players have the same rank (ex aequo), the next rank is skipped.

The final rank when a player finishes is still handeled the same way, separately. We can rename this final_rank, to avoid confusion with the live rank.

In the board UI, the live rank of a player is displayed below his coin count. In the player UI, the live rank is displayed next to his name.
