# v0.14 - Win condition and start over

Each player have a lap count, starting at 1.
When a player reaches the finish line once, the lap count increments.

There is 3 laps in a game.

The game has a ranking array, initialized empty.

When a player finishes his 3 laps, the game adds him to the ranking list.

When all players have finished their 3 laps, the game is over and a leaderboard is displayed, ranking the players (The first is the winner).

The leaderboard has a "start over" button that resets the game in its inital state.
