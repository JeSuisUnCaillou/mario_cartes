# v1.1.4 - Player color

Players are assigned a color from an array of 8 colors on join. The first player takes the first color, etc.

The helmet svg asset has 2 shapes with the color e10000ff. Replace this color by the player's color.

On the game board, the player's avatar in the racetrack has this color. The player's avatar in the side panel too.

On the player screen, top right next to the name (same place as the Rules modal, but on the right) there is a the player's avatar with his color.
